# Discord Coin Bot

Renderで動かすシンプルなDiscordコイントスBotです。

## 🚀 セットアップ

1. リポジトリをフォークまたはクローンする
2. [Discord Developer Portal](https://discord.com/developers/applications) でBotを作成
3. Bot Tokenを取得し、Renderの「Environment Variables」に設定する:
   - Key: `DISCORD_TOKEN`
   - Value: <Botのトークン>
